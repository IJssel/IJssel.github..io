# ECMAScript 6

声明变量let

  1. let声明变量不存在预解析(即自上而下执行,不存在先解析下面)
  2. let声明变量不允许重复(在同一作用域内)
  3. ES6引用块级作用域，即在一个{}外部访问不到，如for循环中外部能调用
  4. 在块级作用域内部，变量只能先声明再使用

声明常量cost（增加两条）

5.const声明的常量不允许重新赋值

6.const声明的常量必须初始化

变量的解构赋值

数组的解构赋值

var [a,b,c] = [1,2,3];
let [a,b,c] = [,123,];
let[a=123,b,c]=[,123,];	=>123,123	数组默认值

对象的解构赋值

let {foo,bar} ={foo:'hello',bar:'hi'};
let{foo:abc=hello,bar} = {bar:'hi};	=>hello,hi对象默认值

对象属性别名（原来名字无效）

let{foo:abc,bar} = {bar:'hi',foo:'nihao'};
console.log(abc,bar);	=>nihao hi
console.log(foo,bar);	=>underfire	如果有了别名那原来的名字无效

let{cos，sin，random) = Math; 直接将对象绑定

字符串的解构赋值

let [a,b,c,d,e] = "hello";
console.log(a,b,c,d,e);	=>a  b  c  d  e
console.log(a,b,c,d);	=>a  b  c  d	省略后面字符

​ 获取长度

let{length} =”hi“；	=>2
let[a,b,c,d,length]="hello";	X错误

字符串相关扩展

includes 判断字符串中是否包含指定的字符串（有则true，否false）

参数1：匹配的字符串；参数2：从第几开始匹配

startswith 判断字符串是否以特定的字串开始

endswidth 判断字符串是否以特定的子串结束

模板字符串

反引号表示模板，模板中的内容可以有格式，通过${}方式填充

let tag=‘<div><span>'+obj.username+’</span> <span>'+obj.age+</span></div>'

let tpl=`

	<div>
	    <span>${obj.username}</span>
        <span>${obj.age}</span>
	</div>

函数扩展

  1. 参数默认值

    function foo(param){	//以前做法
	let p = param||'hello';
	console.log(p);
}
foo('hi');

function foo(param='nihao'){	//当前做法
	console.log(param);
}
foo('hello');

  2. 参数解构赋值

    function foo(uname='list',age=12){	//以前
	console.log(uname,age);
}
foo('zhangsan',13);
//当期
function foo({uname='list',age=12}={}){
	console.log(uname,age);
}
foo({uname:'zhongsan',age:15});

  3. rest参数(剩余参数)

    function foo(a,b,...param){
console.log()	=>1,2,34
}
foo(1,2,3,4);

  4. ...扩展运算符

    function foo(a,b,c,d){
	console.log(a+b+c+d);
}
let arr = [1,2,3,4];
foo.apply(null,arr);	//以前
foo(...arr);
//合并数组
let arr1 = [1,2,3];
let arr2 = [4,5,6];
let arr3 = [...arr1,...arr2];

  5. 箭头函数

    let foo = () => console.log('hello');
foo();
//传多个参数(多个用{})
let foo = (a,b) => {let c=1; console.log(a+b+c);}
foo(1+2);

    注意事项：

    1. 箭头函数中this取决于函数的定义，而不是调用

    2. 箭头函数不可以new

      let foo = () =>{this.num =123;};
new foo();					X错误

    3. 箭头函数不可以使用arguments获取参数列表，可以使用rest参数代替
