# DOM中获取元素的方式

document.getElementById("id的值");根据id获取元素

document.getElementsByTagName("标签的名字");根据元素名字获取元素

document.getElementsByClassName("类的名字");根据的是类样式的名字来获取元素,多个

# JQuery获取元素的方式

根据Id获取--->id选择器--->$("#id的值");一个

根据标签名字来获取---标签选择器--->$("标签的名字"); 多个

根据样式的名字获取--->类型选择器--->$(". 类样式的名字");多个

$function(){
	$("#btn").click(function)(){//获取按钮(根据id选择器,调用点击事件方法
		//id选择器
		$("#dv").text("这是一个有颜色的div");//相当于Dom中的innerText或textContent
		#("#dv").css("backgroundColor","yellow");//设置元素的样式
		//标签选择器
		$("p").text("内容");
		//类选择器
		$(".cls").css("backgroundColor","yellow");
		//多条件选择器
		$("span,p,li,div").css("backgroundColor","yellow");
		//层次选择器
	}
}

层次选择器

偶数选择器:even

奇数选择器:odd

$("#uu>li:even").css("backgroundColor","yellow");//注意计算机0代表偶数,以实际相反
$("#uu>li:odd").css("backgroundColor","link")

层次选择器

eq(3)获取索引的元素

gt(3)索引大于3的所有元素

lt(3)索引小于3的所有元素

$("#uu>li:eq(4)").css("backgroundColor","red");//计算机4,实际为5

# 事件绑定和解绑

为元素绑定事件

bind方法
参数:1事件的名字 2事件处理函数--匿名方法
$("#bth").bind({"click":function(){
},"mouseover":function({
		$(this).css("backgroundColor","red");
	},"mouseout":function(){
		$(this).css("backgroundColor","yellow");
	}});
	
delegate方法	
参数:1要绑定的元素--p 2要绑定事件的名字---click 绑定事件的处理函数---匿名函数
$("#bth").bind({"click":function(){
	$("#dv").delegate("p","click",function(){
	});

on方法(常用手机开发)常用
1.两个参数:事件名字 子集元素
2.三个参数:事件名字 子集元素 事件处理函数(为子级绑定元素)
$("#bth").on({"click":function(){
	$("#dv").append($("<p>标签</p>"));//创建p添加到div
	$("#dv").on({"click":function(){
		alert("点了");
	});
});

# 解绑事件

$("#dv").off("click");
$("#dv").unbind("click");
$("#dv").unadelegate("click");

注意:解绑事件div和p同事解绑(即所有解绑), 自身绑定的事件以及动态绑定的事件都会解绑

​ 如果父级与子集都是通过正常方式绑定事件, 如果通过off解绑.父级被解绑,子级不会解绑

​ 如果子级通过父级调用delegate方式绑定事件, 父级元素使用off解绑事件,这时父级与子级相同事件都会被解绑
