使用腾讯云存储博客图片

为什么好用呢？首先因为它免费，如果是 https 域名的会走公网流量访问会收费，但是收费很少；并且使用腾讯云可以使用客户端进行图片的上传和复制链接，完美支持 https 域名，使用起来非常方便。我个人也是一直在使用腾讯云存储。下面我直接分享一下使用教程，如果有其它好用的图床也欢迎分享哦~

# 官网存储桶的创建

首先在官网首页，我们使用微信账号登录后点击进入控制台

然后在云产品中找到云存储 -> 对象存储

然后点击左侧的存储桶列表 -> 创建存储桶，注意使用这个之前需要进行实名认证，但是可以使用微信或者 QQ 的实名认证，速度非常快，不像其它第三方图床网站还需要上传审核材料。

然后在创建桶的时候，名字随你写，但是权限需要写成公有读私有写权限，那么新建的这个存储桶就可以存放图片了！

# 使用客户端上传和复制图片链接

上面虽然建立好了，但是如果用网页进行上传和复制还是很麻烦的，我们可以直接下载一个客户端进行上传和下载，点击左侧侧边栏中客户端下载进行下载。

然后登录的时候需要 SecretID 和 SecretKey 进行登录。

我们在网页端点击密钥管理去获取这两个值：

然后在页面上找到这两个值并登录进可以进入客户端页面了！

我们可以直接使用拖拽方式将图片上传，然后需要引用图片的时候，点击如下红色框中的按钮，就可以选择复制某一张图片的链接：

想要预览图片的话，可以直接点击下载按钮进行预览（有点麻烦），并且完美支持 https 链接！一般最好是一篇文章一个文件夹，然后把该文章下所有图片放到一起，然后使用 Markdown 语法引用图片就可以了，访问速度非常快并且稳定！需要注意的是，腾讯云不是完全免费的，但是一般博客的访问量较小，收费并不多，5 块钱可以使用后 2 个月左右（根据访问量而定）。

# 其它技巧

由于博客盗图非常多，如果你的图片链接被别人盗用，那么在腾讯云端最后走的还是你的访问流量，这种情况会造成的不必要的流量收费，为了解决这种问题，我们可以使用防盗链技术，只有在你自己设定的域名中访问该存储桶中的图片有效，其它域名无法访问该存储桶中的图片。
